== Server ==
线上版本：http://wallet.xuxinran.me/
=== 架构 ===
基于Flask-RESTful框架，前端配合AngularJS。
=== 需求 ===
* Python 2.7
* pip

=== 初始化服务器 ===
`pip install require.txt -r`
`cd server`
`python createall.py`

=== 启动服务器 ===
* `python run.py`
* 访问 localhost:5000

== 客户端版本 ==
请运行desktop/build/app.exe
